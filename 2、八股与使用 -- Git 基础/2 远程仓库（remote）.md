远程仓库，保存在网络中其它位置（可以是其它服务器上，也可以是本地上）的 Git 仓库。通过远程仓库，能够协作开发。

---
）克隆 `git clone https://github.com/schacon/ticgit` 仓库。
```bash
git clone https://github.com/schacon/ticgit
```
）使用 git remote 查看与当前仓库关联的远程仓库。
```bash
git remote
```
